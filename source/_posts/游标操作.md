##游标在分页中的应用

利用skip和limit函数实现

```
"""
skip()
查询结果中，跳过前9995行
"""

var cursor = db.bar.find().skip(9995);
```

```
"""
limit(N)

限制取出结果，只取前N个
"""

var cursor = db.bar.find().skip(9000).limit(10)
```

```
"""
toArray()

将游标的结果变成一个数组
"""

cursor.toArray()
```
