## Tips

```sh
sed -i 's/http:\/\/cn.archive.ubuntu.com/https:\/\/mirrors.ustc.edu.cn/g' /etc/apt/sources.list
sudo apt update
sudo apt install zsh git ripgrep vim curl build-essential
```
